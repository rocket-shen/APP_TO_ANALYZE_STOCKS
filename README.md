# 📊 APP_TO_ANALYZE_STOCKS

一個結合 React 前端與 FastAPI 後端的財務股票分析面板。

## ✨ 專案特色
- 即時股票資料查詢
- 技術指標分析
- 漂亮的資料視覺化介面
- 前後端分離架構（React + FastAPI）

## 🚀 如何啟動專案

### 1. 前端（React + Vite）
```bash
cd my-app
npm install
npm run dev


```bash
cd server

# 安裝依賴
pip install -r requirements.txt

# 啟動後端
uvicorn main:app --reload --port 8000
